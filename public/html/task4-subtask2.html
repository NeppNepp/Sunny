FlowChartCustomizability

Customizability description:
Step 1: User presses on “Account” button in the left side navigation bar and program pops up a menu.
Step 2: User presses on “Settings” button in account menu and program navigate user to settings page.
Step 3: User presses on “Customize UI” button in the middle of the screen and program pops up “Choose your theme” menu for user to choose their favorite theme.
Step 4: User presses on one of the buttons of the themes in the “Choose your theme” menu and program changes theme of the website.
Files preview description:
Step 1: User presses on “Courses” button in the left side navigation bar and program pops up menu.
Step 2: User presses on a module's button in pop-up menu and program navigates user to that module's page.
Step 3: User hovers mouse cursor over one of the files of the module and program shows a preview of the content of the file.
