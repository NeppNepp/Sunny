Flowchart Threads

To create a new post/thread:
Step 1: The user clicks on the input box to create a post .
Step 2: They will be redirected to the post creation page, where there will be "Title" and "Content" input fields. User enters Title and Content.
Step 3: User clicks on “Post” button.
Step 4: Posts will be processed, archived, and posted to the forum after the moderation period.
To create a new comment:
Step 1: The user clicks on the input box to create a comment.
Step 2: Users can enter their comments, attach files, images or icons.
Step 3: Users press "send" or press Enter to submit a comment. If the comment box is left blank, the user will receive an error message and ask to enter any information before submitting.
Step 4: If submitted successfully, the comment will be saved to the system and posted under the comment section of the main post.
